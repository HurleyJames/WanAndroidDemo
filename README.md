## Introduction

**CodeHub** 是一款使用 WanAndroid、ReadHub 以及 Gank.io 三者 API 开发的技术阅读类的开源项目。项目是基于 MVP + RxJava + Retrofit + Dagger2 + Glide 等框架开发的。

## Features

* WanAndroid、ReadHub 与 Gank.io 的内容。
* 基于 MVP 模式开发。
* 使用了市面上主流的 Android 开发框架。
